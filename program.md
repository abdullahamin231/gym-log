# Program Template

This file is seeded into IndexedDB on first load so it is available offline as a built-in reference.

## Example Program

### Day 1 — Push
- Bench Press — 3 x 8
- Overhead Press — 3 x 8
- Incline Dumbbell Press — 3 x 10
- Triceps Pushdown — 3 x 12

### Day 2 — Pull
- Pull-ups (or Lat Pulldown) — 3 x 8
- Barbell Row — 3 x 8
- Face Pull — 3 x 12
- Dumbbell Curl — 3 x 12

### Day 3 — Legs
- Squat — 3 x 5
- Romanian Deadlift — 3 x 8
- Leg Press — 3 x 10
- Calf Raises — 3 x 12
